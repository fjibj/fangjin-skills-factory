## Skills 列表

### 1. docx-generator

Markdown 转 DOCX Skill for Claude Code - 支持架构图生成和插入。

**功能特性：**
- ✅ Markdown 转 DOCX 专业排版
- ✅ 自动生成架构图（Python matplotlib）
- ✅ 图片自动插入 Word 文档
- ✅ 中文字体支持（SimHei）
- ✅ 目录自动生成
- ✅ 页眉页脚支持

**快速开始：**
```bash
cd docx-generator
npm install
cd scripts/example
python ../generate_arch.py
node ../create-report-docx.js sample-report.md output.docx
```
